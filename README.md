# Tugas 2: Aplikasi Informasi Cuaca (API & TypeScript)

Aplikasi berbasis web yang berfungsi untuk menampilkan informasi cuaca secara real-time dengan memanfaatkan integrasi API pihak ketiga. Projek ini dibangun menggunakan **Ionic Framework** dan **Vite** dengan bahasa pemrograman **TypeScript**.

## Fitur Utama
* **Integrasi API Cuaca:** Mengambil data cuaca terkini secara akurat.
* **UI Komponen Ionic:** Tampilan antarmuka yang responsif dan ramah pengguna (mobile-friendly).
* **TypeScript:** Implementasi kode yang lebih aman dan terstruktur dengan tipe data yang jelas.

## Teknologi yang Digunakan
* [Ionic Framework](https://ionicframework.com/) (Vite)
* TypeScript
* HTML5 / CSS3 / JavaScript
* REST API (Weather API)
